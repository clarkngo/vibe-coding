# Aviation Vibe-Coding Showcase

Small aviation-themed HTML/CSS/JS demos for a beginner vibe-coding workshop. Start at [`aviation/index.html`](aviation/index.html) or open any project file in a browser — no install, no build step.

| Project | File | Category | Description |
|---------|------|----------|-------------|
| Landing Challenge | [`aviation/landing-challenge.html`](aviation/landing-challenge.html) | Game | Timing/reflex game: land the plane in the green zone for points and a streak. |
| Which Aircraft Are You? | [`aviation/aircraft-personality-quiz.html`](aviation/aircraft-personality-quiz.html) | Quiz/game | 5-question personality quiz that maps you to a Cessna, fighter, 747, or glider. |
| Aircraft 101 | [`aviation/aircraft-101.html`](aviation/aircraft-101.html) | Informational | Flip-card encyclopedia of 8 aircraft types with quick facts on the back. |
| Aircraft Anatomy | [`aviation/aircraft-anatomy.html`](aviation/aircraft-anatomy.html) | Study | Click numbered hotspots on a side-view plane to learn each major airframe part. |
| Phonetic Alphabet Trainer | [`aviation/phonetic-alphabet-trainer.html`](aviation/phonetic-alphabet-trainer.html) | Practice tool | Guess the NATO phonetic word for a random letter; includes a full reference table. |
| Pre-Flight Checklist | [`aviation/preflight-checklist.html`](aviation/preflight-checklist.html) | Practical tool | Checkbox checklist with a progress bar that celebrates when you're cleared for takeoff. |
| Flight Tools | [`aviation/flight-tools.html`](aviation/flight-tools.html) | Practical tool | Flight-time calculator plus knots↔km/h and feet↔meters converters. |
| Boarding Pass Profile | [`aviation/boarding-pass-profile.html`](aviation/boarding-pass-profile.html) | Fun/personal | Type your name and dream destination to generate a stylized boarding pass. |
| Cockpit Dashboard | [`aviation/cockpit-dashboard.html`](aviation/cockpit-dashboard.html) | Interactive toy | Clicky switches, altitude slider, and throttle that drive on-screen gauges. |
